# StarNaVs-LA-Hollywood-Themed-E-Commerce-Platform-PHP-MySQL-
A Hollywood-themed e-commerce web app built with PHP and MySQL featuring secure login, role-based dashboards, product &amp; inventory management, shopping cart, and order processing.
This project is a complete database-driven e-commerce system titled “StarNaVs LA”, designed as a Hollywood-themed online shopping and learning platform. It is built using PHP, MySQL, and MVC architecture with a fully normalized relational database (2NF & 3NF).

The system supports role-based access control with separate dashboards for Business Manager, Inventory Manager, Order Fulfillment, Content Editor, and Customers. Core features include secure authentication (bcrypt, prepared statements), product and inventory management, shopping cart, checkout, order processing, receipt generation, shipping tracking, and an educational content module about Hollywood history.

The project demonstrates strong skills in database design (ER modeling, normalization, SQL), backend development, security, and full-stack integration, making it a complete real-world e-commerce application.
